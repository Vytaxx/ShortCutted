About

ShortCutted is a free and open-source shortcut manager website. You might ask, "what's the point if you already have bookmarks?" Well, with ShortCutted, there are public shortcuts people can submit to get added to ShortCutted for anyone to view and use.

ShortCutted is a software from PC Utils which is a collection of software developed by me (and others sometimes).

ShortCutted is completely free to use, just please don't copy the open-source code and distrubute it as your own unless you get explicit premission from me. However, taking the code and modifying it for your own use is completely fine just don't distribute it without my permission.
